# Examples

This directory contains repo-local examples of Noren output.

- [`sample-voice-profile.md`](sample-voice-profile.md) shows the raw Markdown shape of a voice profile: summary, rhythm, named moves, word preferences, anti-patterns, routing metadata, and usage instructions.
- The public website has the polished specimen page with named public writers: https://usenoren.ai/sample-voice-profile

The repo sample is public-safe and abridged. It is based on synthetic/demo writing samples, not on private customer writing.
